À propos de BookNest

Notre mission

BookNest est une plateforme de gestion de bibliothèque personnelle en ligne. Elle permet aux lecteurs de cataloguer leurs livres, de suivre leur progression de lecture et de partager leurs avis avec la communauté.

Fonctionnalités

  • Catalogue commun : une base centralisée de livres enrichie par la communauté
  • Bibliothèque personnelle : gérez votre propre collection de livres
  • Suivi de lecture : marquez vos livres comme « à lire », « en cours » ou « terminé »
  • Favoris : mettez en avant vos livres préférés
  • Notes et commentaires : gardez une trace de vos impressions de lecture
  • Anti-doublon : le catalogue évite les doublons de livres

Technologies utilisées

  • Python 3 — langage de programmation
  • Django — framework web
  • SQLite — base de données
  • HTML5 & CSS3 — interface utilisateur
  • Git & GitHub — versionnement et collaboration
